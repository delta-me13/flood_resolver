# 调洪演算求解器

基于列表试算法的水库调洪演算命令行工具。读取 Excel 格式的水文曲线数据，执行调洪演算，输出含结果图表和过程表格的 Excel 文件。

## Excel 输入文件格式

输入文件必须是 `.xlsx` 格式（不支持旧版 `.xls`），且必须包含以下 **三张工作表**。每张工作表的格式要求一致，缺少任何一张程序都会报错退出。

### 通用规则（三张表均需遵守）

1. **只允许两列**：第一列为横坐标 (X)，第二列为纵坐标 (Y)。不得有多余的第三列。
2. **第一行为表头**：第一行可以是任意文字（例如 "时间"、"流量"），程序会跳过这一行。**表头必须存在**。
3. **从第二行开始填写数据**：每行两个数字，中间不能有空单元格。
4. **数据行数至少为 2 行**（不含表头）。建议提供足够密集的数据点以确保插值精度。
5. **X 列必须单调递增**（从小到大排列）。
6. **所有单元格均为纯数字**，不要包含单位文字（如 "m"、"m3/s" 等）。单位约定见下方各表说明。

### 工作表一：来水过程（洪水过程线）

| 列 A (X) | 列 B (Y) |
|----------|----------|
| 时间 (h) | 入库流量 (m3/s) |

- 表名称必须为：**来水过程**
- 第一行表头，第二行起填数据。
- 时间列从 0 开始。

示例：

| 时间 (h) | 入库流量 (m3/s) |
|----------|-----------------|
| 0        | 100             |
| 1        | 250             |
| 3        | 600             |
| 6        | 300             |
| 10       | 100             |

### 工作表二：水位-库容关系（库容特性曲线）

| 列 A (X) | 列 B (Y) |
|----------|----------|
| 水位 (m) | 库容 (万m3) |

- 表名称必须为：**水位-库容关系**
- 第一行表头，第二行起填数据。
- 水位从小到大排列。

示例：

| 水位 (m) | 库容 (万m3) |
|----------|------------|
| 150      | 500        |
| 155      | 800        |
| 160      | 1200       |
| 165      | 1800       |
| 170      | 2500       |

### 工作表三：水位-泄流关系（溢洪道特性曲线）

| 列 A (X) | 列 B (Y) |
|----------|----------|
| 水位 (m) | 泄流能力 (m3/s) |

- 表名称必须为：**水位-泄流关系**
- 第一行表头，第二行起填数据。
- 水位从小到大排列。

示例：

| 水位 (m) | 泄流能力 (m3/s) |
|----------|----------------|
| 150      | 0              |
| 155      | 50             |
| 160      | 200            |
| 165      | 500            |
| 170      | 1000           |

## 各平台运行指南

以下说明假设您已经将可执行文件 `flood_resolver` 和待处理的 Excel 文件放在同一个文件夹中。

**第一步：打开命令行窗口**

各平台打开命令行的方式如下。您只需要将命令粘贴到命令行窗口中，然后按 Enter 键执行。

### Windows

1. 按键盘上的 **Win + R** 键，输入 `cmd`，点击"确定"，即可打开命令提示符。
2. 在命令提示符窗口中，输入 `cd` 加一个空格，然后将您存放程序的文件夹路径拖入窗口（或手动输入），按 Enter。例如：

   ```
   cd C:\Users\您的用户名\Desktop\flood_tool
   ```

3. 执行程序（示例命令，请根据实际情况修改参数）：

   ```
   flood_resolver.exe -i 输入文件.xlsx -z 160 -d 150 -t 0.5
   ```

4. 如果看到 "参数校验通过！准备开始计算..." 说明程序正在运行。运行结束后，结果文件 `output_result.xlsx` 会出现在同一文件夹中。

### macOS

1. 打开 **访达 (Finder)**，进入 **应用程序 > 实用工具**，双击 **终端 (Terminal)**。
2. 在终端窗口中，输入 `cd` 加一个空格，然后将存放程序的文件夹拖入终端窗口，按 Enter。例如：

   ```
   cd /Users/您的用户名/Desktop/flood_tool
   ```

3. 首次运行需要赋予程序执行权限（只需执行一次）：

   ```
   chmod +x flood_resolver
   ```

4. 执行程序（示例命令）：

   ```
   ./flood_resolver -i 输入文件.xlsx -z 160 -d 150 -t 0.5
   ```

5. 如果看到 "参数校验通过！准备开始计算..." 说明程序正在运行。运行结束后，结果文件 `output_result.xlsx` 会出现在同一文件夹中。

### Linux

1. 在桌面或文件管理器中右键，选择 **在终端中打开 (Open in Terminal)**，或从应用程序菜单中找到 **终端 (Terminal)**。
2. 在终端窗口中，输入 `cd` 加一个空格，然后将存放程序的文件夹路径输入（或拖入），按 Enter。例如：

   ```
   cd /home/您的用户名/Desktop/flood_tool
   ```

3. 首次运行需要赋予程序执行权限（只需执行一次）：

   ```
   chmod +x flood_resolver
   ```

4. 执行程序（示例命令）：

   ```
   ./flood_resolver -i 输入文件.xlsx -z 160 -d 150 -t 0.5
   ```

5. 如果看到 "参数校验通过！准备开始计算..." 说明程序正在运行。运行结束后，结果文件 `output_result.xlsx` 会出现在同一文件夹中。

## 命令行参数说明

| 参数 | 简写 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| `--input-excel` | `-i` | 是 | 无 | 输入 Excel 文件的路径 |
| `--output-excel` | `-o` | 否 | `output_result.xlsx` | 输出 Excel 文件的路径 |
| `--initial-water-level` | `-z` | 是 | 无 | 起调水位，单位 m |
| `--dead-water-level` | `-d` | 是 | 无 | 死水位，单位 m |
| `--precision` | `-p` | 否 | `1e-6` | 迭代计算精度 |
| `--max-iterations` | `-m` | 否 | `30` | 最大迭代次数 |
| `--time-step` | `-t` | 否 | `0.2` | 时间步间距，单位小时 (h) |

### 参数校验规则

- 起调水位必须大于死水位。
- 输入文件和输出文件不能是同一个文件。
- 精度、最大迭代次数、时间步长均必须大于 0。

### 完整命令示例

```
flood_resolver -i flood_data.xlsx -o my_result.xlsx -z 165.5 -d 150 -p 0.000001 -m 50 -t 0.1
```

参数含义：读取 `flood_data.xlsx`，起调水位 165.5m，死水位 150m，精度 1e-6，最多迭代 50 次，时间步距 0.1 小时，结果写入 `my_result.xlsx`。

## 输出说明

程序运行成功后会生成一个 Excel 文件，包含以下内容：

### 结果数据表

包含 10 列数据，每一行代表一个时间步：

| 列名 | 说明 | 单位 |
|------|------|------|
| 时间 (h) | 当前时刻 | h |
| 入库流量 (m3/s) | 当前入库流量 I | m3/s |
| 入库平均流量 (m3/s) | 时段平均入库流量 I_avg | m3/s |
| 时段入库水量 (万m3) | 本时段流入总水量 | 万m3 |
| 出库流量 (m3/s) | 当前泄流 q | m3/s |
| 出库平均流量 (m3/s) | 时段平均泄流 q_avg | m3/s |
| 时段出库水量 (万m3) | 本时段流出总水量 | 万m3 |
| 库容变化 (万m3) | 本时段库容变化量 | 万m3 |
| 总库容 (万m3) | 当前总库容 V | 万m3 |
| 库水位 (m) | 当前库水位 Z | m |

### 过程线图表

结果文件中附带一张散点平滑曲线图，包含两条曲线：
- **来水过程线**：入库流量随时间变化
- **泄水过程线**：出库流量随时间变化

## 常见问题

**Q: 运行时提示 "无法解析文件路径"？**

A: 检查输入文件路径是否正确。如果不确定，可以将 Excel 文件和程序放在同一文件夹，然后只输入文件名（如 `-i data.xlsx`）。

**Q: 提示 "表格形状不符合要求"？**

A: 请检查 Excel 的三张工作表是否都恰好为 2 列，且至少包含表头和 2 行数据。

**Q: 提示 "单元格没有有效内容"？**

A: 某个单元格为空或包含非数字内容（如单位文字）。请检查所有数据单元格均为纯数字。

**Q: 计算结果不收敛？**

A: 可尝试减小时间步长 (`-t` 参数，如改为 `0.1`) 或增加最大迭代次数 (`-m` 参数，如改为 `50`)。同时检查输入曲线数据是否足够密集，以确保插值精度。
